Descripción detallada

Clase que representa una cadena (string) sobre un alfabeto específico.

Una cadena es una secuencia finita de símbolos de un alfabeto. Esta clase implementa las operaciones básicas sobre cadenas utilizadas en teoría de autómatas y lenguajes formales, como obtener prefijos, sufijos, inversas, etc.
